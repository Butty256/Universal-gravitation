# 万有引力シミュレータ（gravitation）

![image](image.bmp)

## 環境

C言語+OpenGLの環境であれば動作する（はず）．
BCC+OpenGLの環境の場合は次のコマンドでコンパイルができる．
```
bcc32c main.c wbmp.c -o gravitation.exe
```

## 使い方

|キー|動作                            |
|----|--------------------------------|
|x   |x軸方向に大きくする             |
|X   |x軸方向に小さくする             |
|y   |y軸方向に大きくする             |
|Y   |y軸方向に小さくする             |
|z   |z軸方向に大きくする             |
|Z   |Z軸方向に小さくする             |
|u   |グラフの拡大                    |
|U   |グラフの縮小                    |
|S   |画像の保存                      |
|Esc |プログラムの終了                |

マウスの左ドラッグでグラフの回転できる．
freeglut環境ではマウスのホイールでグラフの拡大，縮小できる．
